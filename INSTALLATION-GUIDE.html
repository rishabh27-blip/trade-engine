  4. Upload all files from the trade-engine-pwa folder

üîß Troubleshooting

‚ùå Can't install on iPhone?

Make sure you're using Safari browser, not Chrome. iOS only supports PWA in Safari.

‚ùå Install button doesn't appear?

Check that:

  • You're accessing via https:// (not http://)
  • You're using Chrome, Edge, or Safari (supported browsers)
  • Try refreshing the page

‚ùå Data disappeared?

Data is stored in browser storage. If you:

  • Clear browser data ‚Üí exports will be lost
  • Use private/incognito mode ‚Üí data won't persist
  • Use different browser ‚Üí won't see same data

Solution: Use the Export button regularly to backup your trade history!

‚ùå Offline mode not working?

First visit requires internet to cache files. After that, it works completely offline.
